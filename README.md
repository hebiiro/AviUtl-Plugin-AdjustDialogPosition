# AviUtl プラグイン - ダイアログ位置調整

ダイアログの表示位置がモニタからはみ出さないように調整します。
[最新バージョンをダウンロード](../../releases/latest/)

## 導入方法

以下のファイルを AviUtl の Plugins フォルダに入れてください。
* AdjustDialogPosition.auf

## 更新履歴

* 1.0.1 - 2022/07/20 ダイアログ上にさらにダイアログを表示したときの問題を修正
* 1.0.0 - 2022/07/12 初版
